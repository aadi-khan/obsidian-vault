# 📚 Obsidian Vault — Clawbot + Adeel

Welcome to our shared knowledge base. This vault syncs via GitHub.

## How it works

- **You** have Obsidian desktop/app on your machine with the **Git Sync** plugin configured
- **I (Clawbot)** write and update notes here in the vault
- **You** pull on your end to see them in Obsidian

## Folder structure

```
daily/       → Daily notes (meetings, progress, thoughts)
projects/    → Project-specific notes (one folder per project)
reference/   → Reusable knowledge, docs, guides
```

## Sync workflow

1. I push notes here when I learn something worth remembering
2. You open Obsidian → pull (Git Sync fetches latest)
3. You open Obsidian → make local notes → push (Git Sync commits + pushes)
4. On next conversation, I pull and have context from your notes too

## First steps

- Install **Git Sync** plugin in Obsidian (Community Plugins → Git Sync)
- Point it to: `https://github.com/aadi-khan/obsidian-vault`
- Use your GitHub credentials

That's it. Let's start building.
