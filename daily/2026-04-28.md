## Project Summary

**Zameen Clone** — Pakistani property portal MVP

### Tech Stack
- Frontend: Next.js 14 + TypeScript + Tailwind + Shadcn/UI
- State: Zustand + React Query
- Maps: Leaflet (open source, no API key)
- Backend: Node.js + Express + Prisma + PostgreSQL
- Auth: JWT + bcrypt
- Images: Cloudinary

### Phase 1 MVP Scope
1. Homepage with hero search (Buy/Rent/Projects tabs)
2. Property listings page with filters
3. Property detail page with photo gallery
4. Basic auth (register/login)
5. Contact agent form

### Database
- users, agencies, properties, property_images, projects, favorites, inquiries

### Key Decisions
- ISR for SEO-friendly pages
- PKR formatting in Lac/Crore (not raw numbers)
- Marla as base unit (1 Kanal = 20 Marla)
- HTTP-only cookie for JWT
- Zod validation shared frontend ↔ backend
